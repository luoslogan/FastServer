## 🔍 如何查找文档

### 按功能模块
- **认证与授权**: `docs/auth/` - 用户登录、RBAC 权限控制等
- **数据库**: `docs/database/` - 数据库迁移、连接管理等

### 按文档类型
- **概览/指南**: 了解整体架构和设计思路
- **流程说明**: 理解系统如何运转
- **使用示例**: 实际代码示例和测试方法

## 📝 文档规范

- 每个功能模块有独立的子目录
- 每个子目录包含 `README.md` 作为索引
- 文档使用 Markdown 格式
- 代码示例使用语法高亮

## 🤝 贡献文档

添加新文档时：
1. 在对应的功能模块目录下创建文件
2. 更新该目录的 `README.md` 索引
3. 更新本文件的文档结构
